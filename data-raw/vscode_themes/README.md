# Visual Studio Code Themes (Modified Versions)

**Note: These theme files are not intended for use in Visual Studio Code.**

These files are modified versions of existing Visual Studio Code themes. The
modifications were made solely to adapt the themes to the JSON standard format
for reference or analysis purposes. If you wish to use the original themes,
please install them directly from the [Visual Studio
